# Task 1 — Understanding AWS & Free Tier

## Goal

Demonstrate understanding of AWS basics and Free Tier usage by answering the following questions in your own words (3–4 lines each).

### Answers

#### Question 1 — What is an AWS account, and why do you need it at this stage?

AWS account help to give access to Amazon Web Services, this enable the account holder to deploy, build and manage cloud applications. It is the identity used for the creation and for controlling various AWS resources. It is needed to set up infrastructure and deploy application and use to securely access AWS services. It is also used to track billing and usage.

---

#### Question 2 — What is AWS Free Tier, and how long does it last?

The Free Tier allows new users to try many AWS services without any cost within a certain usage limits. This includes free access to some resources like EC2, S3, and Lambda. Most of the free Tier offers last for 12 months after the AWS account is created. While some other services are always free or maybe available as short-term trials. If an account exceed the free limit, this may result in charges.

---

#### Question 3 — Name three AWS Free Tier services and their free usage limits.


Amazon EC2: 750 hours per month of eligible instance usage for 12 months.
Amazon S3: 5 GB of Standard storage, plus limited requests and data transfer, for 12 months.
AWS Lambda: 1 million free requests and 400,000 GB-seconds of compute time per month (Always Free).
